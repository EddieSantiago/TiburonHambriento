¿QUE ES UNA VARIABLE?

Una variable en un algoritmo computacional es una posición de memoria donde se puede almacenar información. Por ejemplo, si un programa debe obtener el área de un triangulo, seguramente la base del mismo y su altura se carguen en memoria en dos variables para poder realizar el calculo, el resultado, probablemente también se asigne en una variable luego del calculo para informarlo al usuario. Una variable pueden ser letras, números y símbolos, pero no puede tener espacios, no puede empezar con un numero y no puede ser una palabra reservada como SiEntonces

Tipos de Variable

  • Variable Auxiliar: También se llaman variables temporales, son las que necesitamos para hacer algo de manera temporal.

  • Variable Acumuladora: Sirven para almacenar un numero y sus valores consecutivos, por ejemplo, si empieza por 0, luego tendrá el valor de 1,2,3,4.

  • Variable de Estado: Se utilizan para saber como se encuentra un objeto en cada momento, la mas simple podría ser una variable booleana, donde su valor seria true o false.
