¿QUE ES Y PARA QUE SIRVE EL COMANDO ESCRIBIR?

La función Escribir es un comando en el pseudocodigo que para el que no sepa que es un comando es una orden que se le da a un programa de computadora que actuá como interprete del mismo, para así realizar una tarea especifica, en este caso el comando Escribir permite mostrar algún valor o dato en la pantalla basándose en la estructura de Escribir 'Ingrese el numero de estudiantes de su grado', con este comando estaríamos mostrando en pantalla lo que este entre comillas.
