¿QUE ES CRUD?

Crud, es uno de los términos mas utilizados por los programadores, simplemente crud, es la abreviación de create (crear) donde toma la letra c, read (leer) donde toma la letra r, update (actualizar) donde toma la letra u y delete (borrar o eliminar) donde toma la letra d, estas operaciones hacen referencia a la interacción de la aplicación con la base de datos, esto en phpMyAdmin se justifica como las opciones de Insert (Insertar datos en bases de datos), Select (realizar búsquedas en bases de datos), Update (Actualizar datos de una base de datos) y Delete (Eliminar datos de una base de datos).
