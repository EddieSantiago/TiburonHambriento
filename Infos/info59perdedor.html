¿QUE ES PhpMyAdmin?

phpMyAdmin es una herramienta, que permite acceder a las funciones de la base de datos MySQL, mediante una interfaz web mas intuitiva, ofreciéndonos entre las principales características operaciones básicas en base de datos MySQL ejemplo crear o eliminar base de datos, es una herramienta fácil de usar, permite optimizar y reportar tablas y la búsqueda en las bases de datos, entre otros.
