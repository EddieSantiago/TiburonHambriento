¿QUE ES UN ENLACE COVALENTE?

El enlace covalente, a diferencia del enlace ionico, se caracteriza dado a que cuando los elementos se unen para formar un enlace en vez de robar electrones o de transferir, se van a compartir electrones, porque los elementos que pueden interactuar en este enlace son no metales, ademas en este tipo de enlace no se forman iones, es decir no existen cargas, ademas Para que un enlace covalente se genere es necesario que la diferencia de electronegatividad entre átomos sea menor a 1,7.




Tipos de Enlaces Covalente

  • Enlace Apolar: El enlace no posee electronegatividad y posee un tipo de molécula diatomica es decir la unión de 2 átomos iguales.

  • Enlace Polar: Se da entre no metales, pero en este caso si hay electronegatividad aunque muy poca.

  • Enlace Coordinado: Se da generalmente entre 3 o mas elementos, donde uno de ellos comparte pares o todos los electrones.
