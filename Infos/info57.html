¿QUE ES MySQL?

MySql es un tipo de sistema gestor de base de datos gratuito, muy fácil de usar y basado en el lenguaje de programación php, el cual se puede utilizar a partir de un sistema local o en el servidor, que tiene como objetivo manipular la base de datos almacenada en diferentes tablas del ordenador.

Tipos de Datos en MySQL

  • Numéricos: Se caracteriza por que sus datos son números ya sea enteros o decimales entre este tenemos tipos como TinyInt, Bit, SmallInt, MediumInt, Integer, BigInt, Float.

  • Fecha: Se caracteriza por que almacena una fecha o una hora, entre estos encontramos a Date, DateTime, TimeStamp, Time y Year.

  • Cadena: Se almacenan datos alfanuméricos, es decir una cadena de caracteres ya sean números, letras o símbolos, entre estos encontramos a Char, Varchar, TinyText, Medium Blob, Set, entre otros.
